# tienda-virtual-php
Este es un proyecto universitario del curso de PHP. Se trata de una tienda virtual que desarrollé por mi cuenta cuando estaba iniciando en el mundo del desarrollo web. Aunque fue uno de mis primeros proyectos, me permitió comprender conceptos clave como manejo de formularios, sesiones, conexión a bases de datos y estructura básica de una aplicación web.

Es un ejemplo sencillo, pero representa una parte importante de mi proceso de aprendizaje y evolución como desarrollador.
